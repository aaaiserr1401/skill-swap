# Платформа обмена навыками SkillSwap

Веб-приложение на Django для обмена навыками между пользователями.

## Возможности

- ✅ Аутентификация и авторизация пользователей (регистрация, вход, выход)
- ✅ Роли пользователей: обычный пользователь, модератор, администратор
- ✅ Каталог и поиск навыков с фильтрацией
- ✅ Профили пользователей с наборами навыков
- ✅ Система обмена навыками с подтверждением обеих сторон
- ✅ Система баллов для мотивации обмена
- ✅ RESTful API с полной документацией (Swagger/OpenAPI)
- ✅ Админ-панель приложения для управления контентом
- ✅ Адаптивный интерфейс (десктоп + мобильная версия)
- ✅ Поиск и фильтрация пользователей
- ✅ Сортировка и пагинация данных
- ✅ Валидация данных и обработка ошибок
- ✅ Логирование и мониторинг (Sentry)
- ✅ Автоматизированное тестирование (unit + integration)
- ✅ Безопасность (хэширование паролей, CSRF, XSS защита)
- ✅ Контейнеризация с Docker
- ✅ CI/CD pipeline (GitHub Actions)
- ✅ Нагрузочное тестирование (k6)

## Требования

- Python 3.8 и выше
- pip (менеджер пакетов Python)
- Виртуальное окружение (рекомендуется)

## Установка

1. **Клонируйте репозиторий**
   ```bash
   git clone <repository-url>
   cd skill-swap
   ```

2. **Создайте и активируйте виртуальное окружение**
   ```bash
   python -m venv .venv
   .venv\Scripts\activate  # Для Windows
   ```

3. **Установите зависимости**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте базу данных**
   ```bash
   python manage.py migrate
   ```

5. **Создайте суперпользователя (администратора)**
   ```bash
   python manage.py createsuperuser
   ```

## Запуск сервера разработки

```bash
python manage.py runserver
```

Затем откройте браузер и перейдите по адресу `http://127.0.0.1:8000/`

## Структура проекта

```
skill-swap/
├── accounts/          # Учетные записи и аутентификация
│   ├── admin_views.py # Админ-панель приложения
│   ├── api_views.py   # REST API endpoints
│   ├── models.py      # Модели данных
│   ├── permissions.py # Кастомные разрешения
│   ├── tests.py       # Unit тесты
│   └── tests_api.py   # Интеграционные тесты API
├── core/              # Основная функциональность
├── skillswap/         # Настройки и конфигурации проекта
│   ├── middleware.py  # Middleware для мониторинга
│   └── settings.py    # Настройки приложения
├── .github/workflows/ # CI/CD pipeline (GitHub Actions)
├── load_tests/        # Скрипты нагрузочного тестирования (k6)
├── logs/              # Файлы логов
├── static/            # Статические файлы (CSS, JS, изображения)
└── templates/         # HTML-шаблоны
    └── admin/         # Шаблоны админ-панели
```

## CI/CD

Проект использует GitHub Actions для автоматизации:
- Запуск тестов при каждом push
- Проверка кода (linting)
- Проверка безопасности
- Сборка Docker образа

## Развертывание

### Docker Compose

```bash
docker-compose up -d
```

### Render / Heroku

Проект готов к развертыванию на Render или Heroku:
- `render.yaml` - конфигурация для Render
- `Procfile` - конфигурация для Heroku
- `build.sh` - скрипт сборки

## Доступные команды

- Запуск тестов: `python manage.py test`
- Запуск интеграционных тестов: `python manage.py test accounts.tests_api`
- Создание миграций: `python manage.py makemigrations`
- Применение миграций: `python manage.py migrate`
- Создание суперпользователя: `python manage.py createsuperuser`
- Сбор статических файлов: `python manage.py collectstatic`
- Панель администратора Django: `http://127.0.0.1:8000/admin/`
- Админ-панель приложения: `http://127.0.0.1:8000/accounts/admin-panel/`
- API документация (Swagger): `http://127.0.0.1:8000/swagger/`
- API документация (ReDoc): `http://127.0.0.1:8000/redoc/`
